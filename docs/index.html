StreamHub-Isotope is a pluggable View for use with StreamHub-Backbone. It will use the jQuery Isotope plugin to lay out Content for a tiled display.

Example Usage

var app = new Hub({
    sdk: livefyreSdk,
    collection: {
        siteId: "303772",
        articleId: "prod0"
    },
    el: document.getElementById("example"),
    view: IsotopeView
}).start();
